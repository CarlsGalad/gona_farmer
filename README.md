3. Order Processing.
4. Order monitoring.
5. Performance chart(Last Six Months and This Week)
6. Local notifications
7. App localizations( English, Hausa, French, Arabic, Yoruba, Igbo)
8. Blog for News.
9. Live chat with customer care.
10. and more.
